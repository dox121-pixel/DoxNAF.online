To play your game here, export it from Unity as a WebGL build:

  1. Open your project in Unity Editor
  2. Go to File → Build Settings
  3. Select WebGL as the platform and click Switch Platform
  4. Click Build and save the output folder
  5. Drop that folder here to play instantly in your browser
How to use
  1. Export your Unity game as a WebGL build (File → Build Settings → WebGL → Build)
  2. Drag the entire output folder (or a ZIP of it) onto the drop zone above
  3. The game loads instantly — no upload, no server, 100% local
  4. Use fullscreen for the best experience

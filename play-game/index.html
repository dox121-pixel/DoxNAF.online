For guaranteed browser playback, export from Unity as a WebGL build:

  1. Open your project in Unity Editor
  2. Go to File → Build Settings
  3. Select WebGL as the platform and click Switch Platform
  4. Click Build and save the output folder
  5. Drop that folder here to play instantly in your browser
How to use
  1. Drop a WebGL build folder or .zip, a zipped standalone build, or a standalone executable (.exe — Windows, .x86_64 — Linux) onto the drop zone
  2. WebGL builds load instantly in an iframe — no upload, no server, 100% local
  3. Standalone executables (.exe / .x86_64) are run via CheerpX (browser x86/x64 compatibility layer — ~80 MB download on first use)
  4. Use fullscreen for the best experience

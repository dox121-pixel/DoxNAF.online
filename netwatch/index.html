✦   What you need
⚠️ Before you run NetWatch, make sure you have:
  • Python 3.7 or newer — free at python.org. Windows users: tick "Add Python to PATH" during install!
  • psutil library — installed with one command (see steps below)
  • A terminal / Command Prompt — already on your computer, no download needed
✦   How to get started
1
Download the file
Click the big green download button above (or below) to save netwatch.py anywhere you like — your Desktop works great.
2
Install Python (if you haven't already)
Go to python.org/downloads, download the installer, and run it. Windows users: on the first screen, tick the checkbox that says "Add Python to PATH" — this is important!
3
Install the psutil helper library
Open Command Prompt (Windows) or Terminal (Mac/Linux) and type:

pip install psutil

Hit Enter and wait a few seconds. Done!
4
Run NetWatch
In your terminal, navigate to where you saved the file (e.g. your Desktop), then type:

python netwatch.py

The tool starts immediately and refreshes every 2 seconds. Press Ctrl+C to quit.

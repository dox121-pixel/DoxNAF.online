✦   What you need
⚠️ Before you run NetWatch, make sure you have:
  • Python 3.7 or newer — free at python.org. Windows users: tick "Add Python to PATH" during install!
  • psutil library — installed with one quick command (see step 3 below)
✦   How to get started
1
Download the app
Click the big green download button above (or below) to save netwatch_app.py anywhere you like — your Desktop works great.
2
Install Python (if you haven't already)
Go to python.org/downloads, download the installer, and run it. Windows users: on the first screen, tick the checkbox that says "Add Python to PATH" — this is important!
3
Install the psutil helper library (once only)
Open Command Prompt (Windows) or Terminal (Mac/Linux) and type:

pip install psutil

Hit Enter and wait a few seconds. You only ever need to do this once!
4
Double-click to open NetWatch
Find the file you saved and double-click it. The NetWatch window opens immediately — no terminal, no typing. Close the window whenever you're done.

# DoxNAF.online

Personal website hosted at [doxnaf.online](https://doxnaf.online).

## Features

- **BlueMap** – Interactive Minecraft map available at [map.doxnaf.online](https://map.doxnaf.online)

## Running Locally

```bash
npm install
npm start
```

Then open **http://localhost:3001** in your browser.

## Tech

- Node.js static file server with BlueMap reverse-proxy
- Hosted on [Render](https://render.com)
